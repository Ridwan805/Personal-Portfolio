The Resturant App

This restaurant menu application, developed using Python's Tkinter library, provides a user-friendly interface for customers to browse and order various food items. It features categories including burgers, pizzas, fries, drinks, rice platters, and combos. Users can view item details, check prices, and decide whether to add items to their order. The app dynamically updates the menu based on user selections and maintains a running total of the order cost. click here see more
